# CRUD React + Node.js

Proyecto ejemplo con arquitectura REST separando el backend (Node.js + Express) del frontend (React + Vite).
<img width="1920" height="1020" alt="image" src="https://github.com/user-attachments/assets/900191d7-2ba9-4605-b7fa-cf192fa40298" />


## Estructura

- `backend/` → API REST en Node.js + Express (almacenamiento en memoria)
- `frontend/` → UI minimalista en React + Vite (colores suaves)

## Cómo ejecutar

1. Iniciar backend:

```bash
cd backend
npm install
npm run dev
```

2. Iniciar frontend:

```bash
cd frontend
npm install
npm run dev
```

Abre `http://localhost:5173`.
