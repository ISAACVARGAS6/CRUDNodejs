# Frontend (React + Vite)

Interfaz minimalista que consume el backend REST de Node.js + Express.

## Instalación

```bash
cd frontend
npm install
```

## Ejecutar

```bash
npm run dev
```

Abre `http://localhost:5173` en el navegador.

> Asegúrate de que el backend (Node.js) esté corriendo en `http://localhost:3000`.
